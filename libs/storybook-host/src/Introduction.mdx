# **FrankJHub Shared UI — Storybook**

### **Overview**
 A unified, reusable component library powering projects under **frankjhub.com**. Every component is **production-ready**, **typed**, and **published to npm** for direct installation across applications.

### **Packages**

- **shared-ui-core** — Framework-agnostic React primitives, utilities, and base components. **Runs standalone**.
- **shared-ui-client** — **Client-only** helpers and interactive components that require the browser (motion, portals, DOM APIs).
- **shared-ui-hero** — Themed components built on **HeroUI** and integrated with **Tailwind**; depends on **@heroui/react**.

### **Key Points**

- **Reusable & versioned** components.
- **Available on npm** for immediate use.
- Designed for **consistent UX**, **faster delivery**, and **shared patterns** across frankjhub.com apps.

### **Installation (choose one)**

- **npm:**  npm i shared-ui-core shared-ui-client shared-ui-hero @heroui/react
- **pnpm:** pnpm add shared-ui-core shared-ui-client shared-ui-hero @heroui/react
- **yarn:**  yarn add shared-ui-core shared-ui-client shared-ui-hero @heroui/react

### **Tech Stack**

- **Tailwind CSS** — utility-first styling and tokens
- **Next.js** — App Router friendly; SSR/SSG compatible
- **Framer Motion** — animations and micro-interactions
- **HeroUI (@heroui/react)** — accessible, themeable UI
- **React Hook Form** — forms and validation
- **TypeScript** — type-safe APIs and props

### **Why This Library**

- **Consistent design system** for all frankjhub.com projects
- **Accelerated delivery** through reuse and shared patterns
- **Typed and documented** components showcased in Storybook
- **Production-proven** in real applications

### **Compatibility**

- **React 18+**
- **Next.js 13+** (App Router recommended)
- **Tailwind 3+**

### **Usage Tips**

- Use **shared-ui-core** for flexible primitives and utilities.
- Use **shared-ui-client** for **browser-only** behaviors and advanced interactivity.
- Use **shared-ui-hero** when you want **HeroUI-styled** components out of the box.

### **Links**

- **Website:** storybook.frankjhub.com
- **npm:** search for **“shared-ui-\*”** packages

### **Notes**
 Ensure **Tailwind** is configured in the host project. For **HeroUI-based** components, install and configure **@heroui/react** as shown above.
